## 5. 车牌规范

### 5.1 牵引车（车头）

格式：`省简称 + 字母 + 5位数字`

```
浙A12345  浙B67890  苏C23456  鲁D78901  皖E34567
沪F89012  赣G56789  闽H12340  鄂I67891  湘J23456
```

### 5.2 挂车（车拍/车挂）

格式：`省简称 + 字母 + 4位数字 + 挂`

```
浙A1234挂  浙B5678挂  苏C9012挂  鲁D3456挂  皖E7890挂
```

### 5.3 字段别名

生成页面时，以下名称可互换使用：

| 标准名 | 别名 |
|--------|------|
| 车牌号 | 车头号、牵引车牌 |
| 挂车号 | 车拍号、车挂牌 |

## 7. 运输场景

| 场景 | 描述 |
|------|------|
| 装货检查 | 装货前车辆检查、货物核验、随车证件查验 |
| 在途监控 | GPS轨迹、车速、罐体温度/压力、异常停车 |
| 异常上报 | 泄漏、事故、超温超压等异常事件 |
| 卸货确认 | 到货验收、签收确认 |
| 空车返程 | 卸货后车辆状态、返程调度 |

## 8. 通用禁止项

- 禁止使用"顺丰"、"中通"、"圆通"等快递公司名
- 禁止使用"淘宝"、"京东"、"拼多多"等电商平台名
- 禁止使用"张三"、"李四"、"王五"等占位姓名
- 禁止使用"某某"、"XX"等占位符
- 禁止使用科技/互联网行业场景（"双十一大促"、"秒杀活动"等）
