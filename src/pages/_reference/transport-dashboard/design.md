# 运输监控看板 — 设计规范

## 布局结构
顶部标题栏 + 时间切换 → 4个指标卡片 → 图表行（趋势图 + 分布图）→ 在途车辆实时表格 → 告警列表

## 组件层级
- PageHeader：标题 + 时间范围切换按钮组
- MetricCards × 4：指标卡片，点击可下钻
- ChartCard × 2：模拟柱状图 + 模拟饼图图例
- RealTimeTable：在途车辆列表（自动30s刷新时间戳）
- AlertList：告警条目列表

## 交互状态

| 状态 | 表现 |
|------|------|
| 加载中 | 各卡片独立展示骨架屏 |
| 空告警 | "暂无告警"文字提示 |
| 指标异常 | 数值变红显示 |
| 温度异常 | 温度值红色加粗 |

## 视觉风格
- 配色：品牌橙黄图表 + 蓝/黄/灰/红状态分色
- 指标卡片：大号数值 28px bold，悬停阴影
- 告警条目：黄色预警背景 / 红色危险背景
- 时间切换：胶囊按钮组
