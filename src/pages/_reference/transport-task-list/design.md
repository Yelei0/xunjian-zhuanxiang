# 运输任务列表 — 设计规范

## 布局结构
顶部标题栏 + 操作按钮 → 筛选卡片（多行筛选字段 + 查询/重置按钮）→ 工具栏（计数 + 批量操作）→ 数据表格 → 底部分页

## 组件层级
- PageHeader：页面标题 + 新建按钮
- FilterSection：6个筛选字段 + 查询/重置
- Toolbar：记录数 + 批量删除 + 导出
- DataTable：13列数据表格 + 排序 + 行选择
- Pagination：分页控件 + 每页条数切换
- ViewModal：任务详情弹窗
- DeleteModal：删除二次确认弹窗

## 交互状态

| 状态 | 表现 |
|------|------|
| 加载中 | 表格区域居中旋转 spinner + "加载中..."文字 |
| 空数据 | 信封图标 + "暂无数据" + 调整筛选提示 |
| 异常 | 红色"数据加载失败" + 重试按钮 |
| 正常 | 表格展示数据，行hover高亮 |

## 视觉风格
- 配色：品牌橙黄 #2563EB + 白底卡片 + 浅灰背景
- 字体：系统字体栈，表格 13px，标题 20px
- 间距：页面 24px/32px，卡片内 20px/24px
- 状态标签：6种状态各有独立配色
- 化学品危险类别标签：3类红、2.3类紫、8类黑
