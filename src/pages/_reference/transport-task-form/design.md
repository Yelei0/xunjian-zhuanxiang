# 运输任务编辑 — 设计规范

## 布局结构
顶部返回 + 标题 → 滚动表单区（4个卡片模块）→ 底部固定操作栏

## 组件层级
- PageHeader：返回按钮 + 页面标题
- FormCard × 4：基本信息 / 化学品信息 / 车辆与人员 / 路线信息
- FormFooter：保存草稿 + 取消 + 提交按钮（sticky定位）
- SubmitModal：提交成功结果弹窗

## 交互状态

| 状态 | 表现 |
|------|------|
| 正常 | 表单默认展示，placeholder 引导填写 |
| 聚焦 | 输入框边框变橙黄 + 外发光 |
| 校验失败 | 输入框边框变红 + 下方显示红色错误文案 |
| 提交成功 | 弹窗展示任务编号和状态流转说明 |
| 取消离开 | 已填写内容时弹出二次确认 |

## 视觉风格
- 配色：品牌橙黄焦点态 + 红错误态 + 灰色只读态
- 卡片分区：白底卡片 8px 圆角，标题带下划线分隔
- 表单网格：2列布局（备注等长字段占整行）
- 底部操作栏：sticky 定位，白底带阴影
